## Bug Fixes

### Fixed `scaffoldEmpty()` Using Incompatible `querySimple()` Method

Fixed `MigrationScaffold.scaffoldEmpty()` generating migration templates that used `db.getClient().querySimple()` instead of `db.query()`. The `querySimple()` method only exists on `PostgresClient` and `BunClient`, causing runtime errors when using `PgClient`.

**Before (broken with PgClient):**
```typescript
await db.getClient().querySimple(`
  -- Your UP migration SQL
`);
```

**After (works with all clients):**
```typescript
await db.query(`
  -- Your UP migration SQL
`);
```

### Files Changed

- `src/migration/migration-scaffold.ts` - Fixed `generateEmptyMigrationFile()` to use `db.query()`
